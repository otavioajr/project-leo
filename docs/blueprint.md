# **App Name**: Alpina Adventures

## Core Features:

- Admin Dashboard: Admin panel for managing events and user registrations, mimicking the flexibility of Aventura Alpina.
- Dynamic Content Blocks: Admin tool to create, edit, and manage content blocks like activities descriptions with details and registration forms.
- Content Management System: System allowing the client to independently update text, images, and event details.
- User Registration Management: Allow the admin to enable or disable the available forms as they see fit

## Style Guidelines:

- Primary color: Forest green (#2C5F2D), mirroring Aventura Alpina's natural aesthetic.
- Secondary color: Adventure orange (#FF6B35) for highlights and action elements.
- Accent color: Sky blue (#4A90E2) for interactive elements and softer contrasts.
- Background color: White (#FFFFFF) to provide a clean and spacious feel.
- Text color: Dark grey (#333333) for readability against the white background.
- Fonts: 'Montserrat' (sans-serif) for headlines and 'Open Sans' (sans-serif) for body text, providing a modern, readable combination.
- Incorporate hero sections with full-width adventure imagery to immediately engage visitors, inspired by Aventura Alpina.
- Utilize card-based layouts for activities, ensuring a clear and visually appealing display of available adventures.
- Spacing: Consistent 24px spacing across the site to maintain visual consistency.
- Corners: Subtle 8px corner radius on elements to soften the interface.
